# X黄评屏蔽器

一个用于 `X.com` 的 Chrome 插件，用来屏蔽评论区、时间线、搜索结果里的疑似涉黄诈骗评论。

它不会直接删除内容，而是将评论做磨砂模糊处理，并提供“恢复查看 / 重新屏蔽”切换，尽量减少打断正常浏览的感觉。

## 功能特点

- 屏蔽 `x.com` 页面中的疑似垃圾黄评
- 覆盖时间线、推文评论区、搜索结果
- 屏蔽后不是删除，而是对头像、昵称、正文进行模糊遮挡
- 支持点击恢复查看，也支持再次重新屏蔽
- 支持手动将某条评论加入整句词库
- 支持自定义短词词库
- 支持删除内置短词、自定义短词、整句词库
- Popup 中快速管理词库
- 独立配置页中以 Tab + 表格方式管理完整词库

## 当前屏蔽规则

插件当前内置了两类规则：

### 1. 文本规则

当一条评论同时满足以下条件时会被自动屏蔽：

- 评论必须是纯文字内容
- 只要带有图片、视频、卡片、引用等非纯文本内容，就直接跳过，不自动屏蔽
- 评论内容包含中文
- 评论长度不超过 40 个字符
- 命中内置短词或自定义短词

默认内置短词包括：

- `哥哥`
- `弟弟`
- `主人`
- `小狗`
- `抱抱`
- `单身哥哥`
- `单身弟弟`
- `求主人`
- `会疼人`
- `领我`
- `认识吗`
- `快来领我`

### 2. 账号特征规则

当一条评论的账号满足以下条件时，也会被自动屏蔽：

- 评论本身仍然必须是纯文字内容
- 昵称命中以下任一关键词时直接屏蔽：`约炮`、`同城`
- 或者昵称中包含 `🌸`，且 `@ID` 以至少 5 位数字结尾

## 手动屏蔽逻辑

对于未自动命中的评论，鼠标悬停时会出现“屏蔽这条”按钮。

点击后会：

- 将该评论加入整句词库
- 自动去掉 emoji
- 之后再次遇到相同整句时直接屏蔽

说明：

- 当前版本不支持手动输入整句词库
- 整句词库主要通过页面上的“屏蔽这条”积累

## 安装方式

### 本地加载

1. 打开 Chrome
2. 进入 `chrome://extensions/`
3. 开启右上角“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择本项目目录

加载完成后，访问 `https://x.com/` 即可生效。

## 使用方法

### 浏览时

- 命中的评论会显示“垃圾评论已屏蔽”
- 点击右上角“恢复查看”可临时展开内容
- 展开后可以点击“重新屏蔽”恢复遮挡

### 手动补充规则

- 将鼠标移到未命中的评论上
- 点击右上角“屏蔽这条”
- 插件会把该评论加入整句词库

### Popup 快速管理

点击浏览器工具栏中的插件图标，可以：

- 查看内置短词数量
- 查看自定义短词数量
- 查看整句词库数量
- 新增自定义短词
- 快速删除部分词条
- 点击 `···` 进入完整配置页

### 完整配置页

完整配置页支持：

- 查看全部内置短词
- 查看全部自定义短词
- 查看全部整句词库
- 删除任意词条
- 通过 Tab 切换不同词库分组

## 技术实现

- Manifest V3
- `chrome.storage.local` 持久化词库
- `MutationObserver` 监听 X 页面动态加载内容
- 内容脚本直接在页面中做规则匹配和模糊遮挡

## 已知限制

- 当前仅适配 `x.com`
- X 页面 DOM 结构如果发生较大变化，可能需要调整选择器
- 当前规则属于启发式规则，不是语义识别，仍可能存在误判或漏判

## 适合谁

- 经常刷 X 评论区，但被黄评诈骗号频繁打扰的人
- 希望保留“可恢复查看”，而不是直接删除评论的人
- 希望自己维护一套屏蔽词库的人
